# CRUD Laravel - Laptops y Marcas

Proyecto de práctica para demostrar habilidades en desarrollo web con Laravel y MySQL.

## 🚀 Tecnologías Utilizadas
- **Laravel 10** - Framework PHP
- **MySQL** - Base de datos relacional
- **Eloquent ORM** - Para relaciones entre tablas
- **Blade** - Motor de plantillas
- **HTML/CSS** - Frontend básico

## 📋 Características
- ✅ CRUD completo (Crear, Leer, Actualizar, Eliminar)
- ✅ 2 tablas relacionales (laptops + marcas)
- ✅ Relaciones Eloquent: belongsTo / hasMany
- ✅ Migraciones de base de datos
- ✅ Validación de formularios
- ✅ Vistas Blade organizadas

## 🔧 Instalación
1. Clonar el repositorio
2. Ejecutar `composer install`
3. Copiar `.env.example` a `.env` y configurar la base de datos
4. Ejecutar `php artisan migrate`
5. Ejecutar `php artisan serve`
